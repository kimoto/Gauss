■概要
ガンマ変更ソフト。画面の明るさを変更できるソフト。
厳密には明るさではなく、モニタデバイスのガンマ値を変更します。
主にゲーマー用途に作成しています。

このソフトウェアの特徴は次の通りです。
・ガンマ値を変更出来る
・キーボードでガンマを変更可能
デフォルトはCtrl+PageUpで明るさアップ、Ctrl+PageDownで明るさダウン、Ctrl+Resetで初期状態(変更可能)
・マルチモニタ環境では、モニタごとにガンマを設定可能
モニタ1とモニタ2に違うガンマ設定ができる。たとえばモニタ2だけ明るくしたいとかときに使う
・デスクトップにショートカットを作成可能(作成したものを別の場所に移動することも可能)

■こだわり
・設定ダイアログがマウスの近くに表示されます
すこしでもマウスの移動距離を減らすことが出来るようにしています

■動作環境
・windows7 32bit/64bit 
※WindowsXPなど、他の環境でも正常に動作すると思われるが確認出来ていません
・グラフィックボードのある環境
グラボがない環境ではガンマの設定が反映されない可能性があります

■今後の方針
・暗部補正機能を実装する
