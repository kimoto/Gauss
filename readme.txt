■概要
ガンマ変更ソフト
・ガンマを変更出来る
・キーボードでガンマを変更可能
・キーボード設定を変更可能
・マルチモニタ環境では、モニタごとにガンマを設定可能
・デスクトップにショートカットを作成可能(別の場所に移動することも可)

■FAQ
Q1. 常駐するけど重いの？
A1. 常駐しますが、これはキー入力のグローバルフックと言って
キー入力されたときに初めて処理が行われ、それ以外の状態では処理は一切行われません。
したがって常駐と言っても常時処理が走るわけではありません。

しかし、キー入力の時だけ処理が走るのであれば軽いのかと言われれば、答えはわからないです。
人間には認識できないレベルの些細な処理時間ではあるが確実に処理は増えている、というのは間違いないです
それは、キーが押されたときにそれが設定されたキーの組み合わせであるかどうかの確認処理が走るためです。
でも、そこにこだわるのであれば他にもいくらでもキーショートカットが設定されたアプリケーションがあるわけで
それらのアプリケーションもすべて終了させたほうが良いかと思われます
